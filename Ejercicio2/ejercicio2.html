Explicacion de terminos y etiquetas

función
Una función es una parte reutilizable de un programa.
let
La instrucción let declara una variable de alcance local con ámbito de bloque(blockscope), la cual, opcionalmente, puede ser inicializada con algún valor.
const
Esta declaración crea una constante cuyo alcance puede ser global o local para el bloque en el que se declara.
Etiqueta a,/a
crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.
Etiqueta dl,/dl>
Es la etiqueta de apertura y cierre que indica que dentro viene una lista descriptiva.
Etiqueta footer,/footer
Usadas para definir el pie de página.
